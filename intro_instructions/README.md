## Learning Outcomes

### What you should learn by the end

- What an FPGA is and how it works.
- How to read and write Verilog to describe digital circuits.
- How to use truth tables to verify digital circuits.
- How to implement combinational logic including full adders.
- How to drive seven-segment displays from FPGA switches.


## Required Materials

- FPGA development board
- USB cable for programming
- Project files from this repository
- Intel Quartus Prime Lite Edition installed. See [Quartus Install](quartus_install.ipynb) for instructions.
